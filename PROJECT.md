## What OfferAIO is
OfferAIO helps students auto-apply to Summer 2027 internships **in their own
browser**. It watches live internship postings, fills each application on the
major applicant tracking systems, drafts cover letters in the user's voice, and
lets the user review and submit. Positioning: "Auto apply to internships while
in class."

## Deploy process (IMPORTANT — how changes go live)
1. Edit files locally, then **commit to `main`** on GitHub. Easiest via the web
   "Upload files" flow: `github.com/TobyBerg43/OfferAIO/upload/main[/<subdir>]`.
2. GitHub Pages rebuilds automatically (~1 min).
3. **Purge the Cloudflare cache** so changes serve immediately:
   dash.cloudflare.com > offeraio.com > Caching > Configuration > **Purge Everything**.
   (New URLs don't need a purge; changed existing URLs do.)
- Large files (landing.html, OfferAIO.html) are edited with **exact-match patch
  scripts** (Node/`String.replace`, fail loudly if a target is missing) rather
  than hand-retyping, then re-uploaded. There's also a manual-dispatch
  "Patch dashboard" GitHub Action pattern (`patch.yml` + `patch_dashboard.js`).

## The Chrome extension (`extension/`)
- Manifest V3. Name "OfferAIO — Auto Apply", v1.0.0.
- Files: `manifest.json`, `popup.html`, `popup.js` (logic — must stay external, MV3
  blocks inline scripts), `content.js` (the cross-ATS form filler), `bridge.js`
  (relays the profile from offeraio.com into the extension), `icons/16,48,128`.
- Fills fields across Greenhouse, Lever, Ashby, Workday, SmartRecruiters, iCIMS,
  Workable, Jobvite, BambooHR, Breezy, Taleo, Handshake, LinkedIn, ZipRecruiter,
  Indeed, Wellfound — by matching autocomplete/name/label attributes.
- Safety: runs on the user's IP/session; user reviews + submits; **CAPTCHAs never
  bypassed**; resume upload stays manual (browsers forbid scripts attaching files).
